TYPED. Kirby Theme

TYPED. is a blog theme for Kirby CMS. If you've never heard about Kirby, it's an awesome Flat CMS that's super powerful, super easy, and super fast. You may want to read more about Kirby on the Get Kirby website. It's free to try out and you'll need to buy a license if you want to use it on a live website. It's worth it :)

Introduction

Built for version: Kirby 2.3. It is based on the Strongly Typed theme by HTML5 UP and was adapted for Kirby by HIWEB.

Features

The full power of Kirby for an awesome feature-packed blogging theme:

  • The nav and header are fully customizable from the panel
  • The 3 column layout called "services" can be changed from 2 to 4 columns
  • The banner section includes an image selector, text color picker, and the text field.
  • The latest blog posts are displayed on the homepage. But you can change the order!
  • The number of posts displayed in the blog page is customizable.
  • Smart pagination is included
  • Each blog post has its own page with optional cover image.
  • A post can be published "in the future" with a custom Published Date feature.
  • The sidebar is optional and can be displayed on either side.
  • Customize the number of posts in the sidebar. It's a smart sidebar that won't ever include the posts that are already visible in the main section.
  • The contact form is optional. It's powered by formspree.io and has built-in AJAX. It's fully customizable. You can even change your email address from the panel!
  • The information details are also optional. Add as many contact information or social links straight from the panel.
  • The footer is obviously also customizable!

Pages included:

  • Homepage
  • Blog Page
  • Article Page
  • Error Page
  • Blank Page

Installation

  1. Buy the theme on Gumroad or on Creative Market
  2. Download the theme. You'll get a .zip archive that contains your full website in just a few megabytes (with demo content).
  3. Upload it to your server and unzip it. Alternatively, unzip it locally and start a local server with php -S localhost:8000 (requires php installed).
  4. Connect to the pannel at yoursite.com/panel or locally at localhost:8000/panel Using the default username: adminand password: pass
  5. Change the admin password right away, change the site settings, create your own content... do your thing!
